# Asiago

## Dependencies
 - Docker (for building)
 - Make (for running scripts)
 - QEMU (for Aura execution)
 - that's it!

## Getting
```
$ git clone --recurse-submodules git@github.com:projectasiago/asiago.git
```

## Building (you may skip this step)
```
$ make
```

## Running
```
$ make run-aura
```
